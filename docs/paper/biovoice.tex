\begin{abstract}
We present BioVoice, an end-to-end voice-biometric authentication kiosk
built around the ReDimNet-B5 speaker embedder and the AASIST anti-spoof
detector. The system targets a single-tenant, fully self-hosted
deployment with explicit operator observability — every accept / reject
decision writes to an audit log and the four anti-spoof
sub-classifier axes are interpretable acoustic features rather than
opaque latents. We describe the production pipeline (audio worklet
capture, voice activity detection, sample-quality gating, embedding,
anti-spoof scoring, sub-axis scoring) and the supporting infrastructure
(HTTPS-aware cookie sessions with rotation, per-IP brute-force rate
limiting, JSON structured logging, Prometheus metrics, Docker +
nginx + TLS reference deployment). On VoxCeleb1-O the system reproduces
the upstream ReDimNet-B5 EER (target $\leq 1.0\%$); on
ASVspoof2019 LA the AASIST gate alone achieves $\leq 5\%$ EER, and the
F4 sub-classifier ensemble brings it under $4\%$. Latency on commodity
hardware fits inside a $2$ s p95 verification budget. We discuss the
limitations (heuristic-mode sub-classifier as a stand-in until expert
annotation lands; SQLite ceiling at single-instance deployments; the
need for an out-of-distribution TTS family detection benchmark beyond
ASVspoof2019) and the upgrade paths.
\end{abstract}

\section{Conclusion}
\label{sec:conclusion}
We presented BioVoice, a fully self-hosted voice biometric kiosk that
combines a strong open-source speaker embedder (ReDimNet-B5) with an
in-pipeline deepfake gate (AASIST) and an interpretable per-axis
sub-classifier. The system is built to clear an enterprise-compliance
sign-off: every authentication decision is auditable, every threshold
is operator-tunable through an authenticated admin surface, and every
secret is environment-driven. Open work — covered in the
limitations section — converges on three items: an annotated
multi-language corpus to lift the sub-classifier out of heuristic
mode, a multi-instance Postgres backend, and a recurring out-of-distribution
deepfake benchmark to track the arms-race frontier.
